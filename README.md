EYEQMC Study Portal project
===========================

## 제작 목적
* 매주 진행되는 스터디의 효율적인 관리를 위해 제작.
* 앞으로 진행될 스터디의 목록을 제공.
* 스터디원의 커뮤니케이션을 위해 게시판도 운영.
* 스터디 장소 확인을 위해 지도 제공.
